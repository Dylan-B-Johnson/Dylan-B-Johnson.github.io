Assignments

Assignment 1:
Basic HTML

Demonstrates basic usage of a HTML including divs, paragraphs, headers, images, hyperlinks, and tables.

Assignment 2:
Basic CSS

Demonstrates basic usage of CSS, including padding, margin, coloring, and borders.

Assignment 3:
Page Layout

Demonstrates the usage of flex-boxes and media queries to make a layout that looks good on mobile and desktop

Assignment 4:
Recreate CSS Page

Demonstrates using CSS to create a complex, professional looking layout, by attempting to clone an existing USC webpage.

Assignment 5:
JavaScript: Buttons, Functions, and More

Demonstrates basic JavaScript through usage of onclick listeners, functions, variables, and editing HTML element content and styleing.

Assignment 6:
JavaScript: Conditionals

Demonstrates basic JavaScript conditionals to modify the state of the webpage based on the input.

Assignment 7:
JavaScript: Loops

Demonstrates the use of for loops to generate a lot of HTML elements using JavaScript.

Assignment 8:
JavaScript: Arrays

Demonstrates the use of an associative array to add a number of HTML elements in JavaScript.

Assignment 9:
JavaScript: Classes

Demonstrates the use of classes to encapsulate a variety of properties about an element and populate them at runtime.

Assignment 10:
JavaScript: Parsing JSON

Demonstrates the creation of HTML content from JSON files at runtime.

Assignment 11:
Node: Hello World and Render Deployment

Demonstrates a simple web service using Node and its deployment to Render. Render URL. Code.

Project Parts

Part 1:
Topic

Contains a description of the project idea, pages it will have, and the type of data it will use on its backend.

Part 2:
Wireframe

Demonstrates the basic design of the website using Mockitt, as well as the features it will have, without worrying about color scheme and themeing.

Part 3:
HTML CSS Wireframe

Demonstrates the basic design of the website in HTML and CSS, as well as some of the features it will have, without worruing about color scheme, images, themeing, and javascript.

Part 4:
Colors, Pictures, and Text

This is the HTML CSS Wireframe, but proper colors, pictures, and text have replaced the placeholders. Nav menues and a hamburger icon have also been implemented.

Part 7:
Forms

Demonstrates the use of an HTML form to submit a POST request to a email service and the use of an iframe to embed content.

Part 8:
React

Demonstrates react by converting the entire site into react components and deploying it. GitHub Pages deployment. Code. Render backend. Backend code.
