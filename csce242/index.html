Assignments

Assignment 1:
Basic HTML

Demonstrates basic usage of a HTML including divs, paragraphs, headers, images, hyperlinks, and tables.

Assignment 2:
Basic CSS

Demonstrates basic usage of CSS, including padding, margin, coloring, and borders.

Assignment 3:
Page Layout

Demonstrates the usage of flex-boxes and media queries to make a layout that looks good on mobile and desktop

Assignment 4:
Recreate CSS Page

Demonstrates using CSS to create a complex, professional looking layout, by attempting to clone an existing USC webpage.

Assignment 5:
JavaScript: Buttons, Functions, and More

Demonstrates basic JavaScript through usage of onclick listeners, functions, variables, and editing HTML element content and styleing.

Assignment 6:
JavaScript: Conditionals

Demonstrates basic JavaScript conditionals to modify the state of the webpage based on the input.

Assignment 7:
JavaScript: Loops

Demonstrates the use of for loops to generate a lot of HTML elements using JavaScript.

Assignment 8:
JavaScript: Arrays

Demonstrates the use of an associative array to add a number of HTML elements in JavaScript.

Assignment 9:
JavaScript: Classes

Demonstrates the use of classes to encapsulate a variety of properties about an element and populate them at runtime.

Part 1:
